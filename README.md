轻量级 LSPosed 模块，专为 **海阔视界** 与 **嗅觉浏览器** 设计。

本模块可移除应用内置的拦截文件检测校验，解除功能限制，从而提升使用体验。

---
✨ 模块功能：
 -
· **自动激活**：进入应用后自动完成无限制激活，并显示明确的 Toast 提示。

· **解除限制**：全面拦截应用内的导入检测、过滤及校验逻辑，移除导入文件、小程序、脚本、分享等风险禁止限制。

· **零干扰：** 模块运行完全不影响应用原有功能与使用体验。

📱 适用环境：
-
1. 已 ROOT 设备：**LSPosed** 框架 - 勾选作用域
2. 未 ROOT 设备：**LSPatch** 框架 - 修补APK

· 目标应用：海阔视界、嗅觉浏览器（含共存版）
· 包名匹配规则：

  · **原版应用：** 包名自动适配。
  
  · **共存版包名：**
  
    ·[✓]海阔视界：包名以 com.example.hike 开头即可
    （例如 com.example.hike666、com.example.hikeabc）
    
    ·[✓]嗅觉浏览器：包名以 com.hiker 开头即可
    （例如 com.hiker.666、com.hiker123）
